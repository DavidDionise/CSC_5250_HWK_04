******** COMPILING *********

To compile the program, navigate to the root directory (Dionise_David_HWK_4) and enter the command "make" (excluding parenthases).

******** RUNNING ***********

After compiling the program, still in the root directory, enter the command "./barber <num of customers>".

For example, to simulate 50 customers, enter the following command :

	./barber 50
